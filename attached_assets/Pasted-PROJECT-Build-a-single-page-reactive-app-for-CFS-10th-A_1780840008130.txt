PROJECT
Build a single-page reactive app for "CFS 10th Anniversary Giveaway" on diaasadek.com — 3 winners, drawn live via a "Cipher Wheel" mechanic.

AESTHETIC CONSTRAINTS (NON-NEGOTIABLE)

Dark mode only. Background #09090D. Surface #111318.
Electric neon blue #00B4FF as primary accent. Gold #D4A843 for rank-1 highlights.
Fonts: Cairo (Arabic), Orbitron (tech/numbers), Barlow Condensed (labels).
Glassmorphism on modals only. Subtle scanline overlay on body.
Clip-path angular cuts on cards/buttons (no round corners).
No emoji in UI chrome. No Bootstrap. No Tailwind. No generic purple gradients. No rounded cards.
Corner-bracket decorations on panels (CSS only).
All Arabic text dir="rtl", perfect typographic rhythm.


PAGE LOGIC — 4 TIME-BASED STATES (Cairo time, GMT+3)
All transitions happen client-side without page reload, driven by setInterval.
StateWhenLabel1 — STANDBYBefore Oct 10, 9:30 PMCountdown timer2 — GATHERINGOct 10, 9:30–10:00 PMPre-event dashboard3 — LIVE DRAWOct 10, 10:00 PMCipher Wheel drawing4 — RESULTSAfter draw completesWinners stage
Add ?state=2, ?state=3, ?state=4 URL params to force any state for testing.

STATE 1 — STANDBY

Large Arabic heading: الذكرى العاشرة — مسابقة السحب الكبير
Metallic countdown: days / hours / minutes / seconds, Orbitron font, angular boxes with top-gradient and horizontal midline.
Info grid: date, platform, winner count, mechanic.


STATE 2 — GATHERING

Gold live indicator: GATHERING — يتجمع المتسابقون
Participant search box (full-width): user types their name → matching cards highlight in cyan + auto-scroll. If no match → red message + direct link to WhatsApp Channel: https://www.whatsapp.com/channel/0029Vb6jrI44yltQQfvkg41o
Participant grid (all names below, deduplicated).
Two CTA buttons: هل أنت متحمس؟ ⚡ (pulse on click) + دخول فيديو المسابقة 🎬 (opens registration flow).
Large green WhatsApp CTA button linking to the channel above.
Registration flow:

If logged in → modal: الحساب موثق. أنت جاهز للسحب.
If not → glassmorphism modal with email input + توثيق واشتراك button.




STATE 3 — LIVE DRAW

Red live indicator: LIVE DRAWING — السحب الآن
Cipher Wheel: 4 concentric metallic rings (CSS animation, no canvas), rotating at different speeds. Center: CFS in Orbitron.
Below rings: rapidly cycling username display (fast → slow → freeze on winner).
"Start draw" button. Press 3 times for 3 winners. Each pick adds an animated row below the machine (rank badge + username + Arabic rank label).
After 3rd pick: button changes to عرض الفائزين → triggers State 4.


STATE 4 — RESULTS (Winners Stage)

Desktop: 3-column podium order (2nd | 1st | 3rd). Mobile: vertical, 1st on top.
Each card:

Rank badge (gold/silver/bronze border + color)
Circular character image frame (rotating ring, placeholder SVG person icon, Arabic label صورة البطل)
Username (IGN), CFS ID placeholder
Arabic rank label


1st-place card: gold top-border glow + subtle gold background tint.
